# Register Discord-Roblox Link

<primary-label ref="bot-command"/>
<secondary-label ref="staff-only-command"/>

Links a Roblox account to a Discord account permanently.

> This can ONLY be run by a <tooltip term="bot-admin">bot
> administrator</tooltip> as part of the [Staff Linking process](Staff-Links.md).

{style="warning"}

## Usage

- `/staff-roblox-link <discord: Discord User> <roblox: Text>`

### Parameter Details:

- [`<discord: Discord User>`](Parameter-Types.md#discord-user) The Discord user to link.
- [`[roblox: Text]`](Parameter-Types.md#text) The *current username* of the Roblox account to link to the given
  Discord account.
