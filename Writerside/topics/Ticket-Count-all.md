# Ticket Count (all)

<primary-label ref="bot-command"/>
<secondary-label ref="staff-only-command"/>

Counts the number of bans done by *all* staff members for the current/last pay period.

## Usage

- `/ticketcount-all [lastweek: Boolean]`

### Parameter Details:

- [`[lastweek: Boolean]`](Parameter-Types.md#boolean) If true, the count will be run for the _last_ pay period instead
  of
  the current one. Defaults to `False`.
