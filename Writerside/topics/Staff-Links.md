# Staff Links

In order to use staff-only commands, the executor must pass the following checks:

- Must have a staff role in Discord.
- Must have their Roblox account linked to Discord. This can only be done by a <tooltip term="bot-admin">bot
  administrator.</tooltip>, so please ask one of them to be link you if you're not yet.
- Must have the appropriate permissions in-game which is dependent on the action being performed. If you can do it
  in-game, you can do it with the bot.

## Linking Procedure

Make sure the <tooltip term="bot-admin">bot administrator</tooltip> you're contacting has your Roblox username or ID.
Double-triple-check to make sure it's right! After that, the [`/staff-roblox-link`](Register-Discord-Roblox-Link.md)
command can be used to perform the linking.
