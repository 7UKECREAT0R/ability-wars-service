# Set Punches

<primary-label ref="bot-command"/>
<secondary-label ref="staff-only-command"/>

Sets the punch count for a given player. Must have in-game permissions to set punches.
After running, the success message will show both the old and new punch counts.

## Usage

- `/aw-setpunches <target: Player> <punches: Integer>`

### Parameter Details:

- [`<target: Player>`](Parameter-Types.md#player) The player to set the punch count of.
- [`<punches: Integer>`](Parameter-Types.md#integer) Their new punch count.
