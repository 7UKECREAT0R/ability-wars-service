# Ban Player

<primary-label ref="bot-command"/>
<secondary-label ref="staff-only-command"/>

Bans a player in-game permanently.

## Usage

- `/aw-ban <target: Player> <reason: Text>`

### Parameter Details:

- [`<target: Player>`](Parameter-Types.md#player) The player to ban.
- [`<reason: Text>`](Parameter-Types.md#text) The reason for the ban. All bans must have some kind of reason attached,
  even if it's short.
